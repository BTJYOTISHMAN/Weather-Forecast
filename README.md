# 🌦 Advanced Weather Forecast App

A professional weather forecasting web application built using HTML, CSS, and JavaScript.

## 🔥 Features
- City weather search
- Use my location (GPS)
- Hourly temperature forecast
- 7-day weather forecast
- Real weather icons
- Google Maps integration
- Dark mode
- Responsive design

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- OpenWeather API
- Google Maps Embed

## ▶ How to Run
1. Download the project
2. Open `script.js`
3. Add your OpenWeather API key
4. Open `index.html` in browser
